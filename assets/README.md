# Assets
Contains customised versions of externals assets to reduce page size.

## Bootstrap
Includes only the components used by the site. Can be generated by
running:

```bash
scss --style compressed custom.scss > custom.css
```
where custom.scss contains:

```sass
@import "functions";
@import "variables";
@import "mixins";
@import "reboot";
@import "type";
@import "images";
@import "grid";
@import "card";
@import "utilities";
```

This generates a css file that's 52kB as opposed to the default bootstrap.css
file that's 142kB.

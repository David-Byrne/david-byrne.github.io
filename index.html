Web

I have made a few websites both for assignments and to improve my skills when I've got free time. They're getting better each time and I learn more and get more experienced at it.

I made The Working Title in a group project at the end of first year in our C programming module. Unfortuantely it doesn't do much at the moment as its back-end was built using C and CGI which isn't supported by the server it's currently on.

Over the Christmas holidays in first year I built Crypto to improve my cryptography skills. It supports Reverse, Rot13 and Affine cyphers but was more built as a proof of concept rather than to serve a cryptography service.

During the first semester of first year I created Byrne's Brilliant Bazaar for the continous assessment part of the module. We were given a set list of things to include each week and we were encouraged to personalise it and give it a light humoured theme.

C

I did C in first year and you can see many of the labs and some of my projects on my C GitHub repository.

We finished the year with a final group project. My team decided to use CGI to connect it to a webpage for a GUI (see The Working Title in the web section). We encorperated the javascript we learnt in other modules as a client-side language as we weren't limited to just C for the project. The back end was written entirely in C and it kept track of stock, decrypted the card details, dealt with all the transactions, saved the details entered in feedback forms into text files and allowed people with admin access to edit and delete stock records.
At that point it was the largest single project either of us had ever done and our first time to use CGI and C in a non-trivial manner. We found it exteremly satisfying however and learned loads from the experience.

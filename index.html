Over the Christmas break in second year I developed Hangons. It's a fully client side web-app to parse the Hangouts.json data file you get when you back up your Hangouts messaging data. It uses Angular.js (1) and a Bootstrap theme and can convert the Hangouts.json file into a much more friendly format. It can create a simplified version of the JSON file which takes up less than a fifth of the space compared to the version offered by Google Takeout.

C

I did C in first year and you can see many of the labs and some of my projects on my C GitHub repository.

We finished the year with a final group project. My team decided to use CGI to connect it to a webpage for a GUI (see The Working Title in the web section). We incorporated the javascript we learnt in other modules as a client-side language as we weren't limited to just C for the project. The back end was written entirely in C and it kept track of stock, decrypted the card details, dealt with all the transactions, saved the details entered in feedback forms into text files and allowed people with admin access to edit and delete stock records.
At that point it was the largest single project either of us had ever done and our first time to use CGI and C in a non-trivial manner. We found it extremely satisfying however and learned loads from the experience.
